# Exercise 3.03

Publishes a git repository into dockerhub

Usage

    sudo ./publish.sh <GITHUB REPOSITORY> <DOCKERHUB REPOSITORY>

Example

    sudo ./publish.sh JanneJP/Devops-with-Docker-2023-Exercise-3.03 jannejp/devops2023-3.03:latest
